Comandos para Users

Qualquer user pode usar esses comandos

Comandos O que faz?
?help, ?ajuda ou ?hp Retornar uma lista de comandos disponíveis.
?eu Retornar informações do seu usuário.
?sobre @*user* Retornar informações dos outros usuários.
?myinvite ou ?miv ID Cria um link o canal referente a ID. Se ID for vazio, cria um link do canal atual.
?ping Retornar o tempo médio de resposta do bot.
?mat, ?math ou ?m operação Retornar o resultado de uma operação matemática.
?arcade Verifica se há algum jogo em fase beta.
?dicio, ?dic ou ?dc "palavra" Procura a definição ou classificação de uma palavra.
?staffajuda "texto" ou ?sta "texto" Envia uma mensagem ao canal de staff. Enviar entre as aspas!
?meajude ou ?ma Ajude os criadores do bot.
?botinvite @bot ou ?bi @bot Cria um link de convite para um bot.
[game] ?dice Jogo de dados - ganha mais pontos se todos os dados forem iguais.
[game] ?pedir Pede seus edinaldos iniciais (moedas do jogo).
[game] ?desafiar @user ou ?de @user Desafia um usuário no jogo da velha.
[game] ?play número ou ?p número Marca uma posição no jogo da velha. Os números variam de 1 a 9.
[game] ?desistir ou ?dr Desiste do jogo da velha.

Comandos para Staff

Os staffers devem ter permissão de "gerenciador de canais"

Comandos O que faz?
?clear número Apagar uma determinada quantidade de mensagens.
?clear_all Apagar todas as mensagens de um canal.
?invites ou ?iv Lista os invites dos usuários.
?lock @role id-channel Bloqueia um canal, exceto para "@role". Se "id-channel" não for informado, o canal atual será bloqueado. Você também pode usar somente ?lock para bloquear para todos.
?unlock @role id-channel Desloqueia um canal, exceto para "@role". Se "id-channel" não for informado, o canal atual será bloqueado. Você também pode usar somente ?unlock para desbloquear para todos.
?defstaff #channel-name ou ?dfc #channel-name Define um canal para os user solicitarem ajuda da staff.
?udefstaff #channel-name ou ?udfc #channel-name Remove o canal de ajuda da staff.
?defin #channel-name ou ?dfi #channel-name Define o canal de entrada de usuários.
?udefin #channel-name ou ?udi #channel-name Define o canal como não sendo de entrada de usuários.
?defout #channel-name ou ?dfo #channel-name Define o canal de saída de usuários.
?udefout #channel-name ou ?udo #channel-name Define o canal como não sendo de saída de usuários.
?channelvname #channel "novo-nome" ou ?chvn #channel "novo-nome" Renomeia um canal de voz para novo-nome.
?channeltname #channel "novo-nome" ou ?chtn #channel "novo-nome" Renomeia um canal de texto para novo-nome.
?statuschannel ou ?sch Cria canais de status, que mostram o número de usuário e bots em um servidor.
?udefstatus ou ?uds Remove os canais de status.
?sorterole @cargo ou ?sr *@cargo* Sorteia um usuário que um determinado cargo.

Comandos para Admin

Os users devem ter permissão de "Administrador"

Comandos O que faz?
?allmembers ou ?am Lista a quantidade de usuários e bots no server.
?ban @user Banir um usuário.
?kick @user Kicka um usuário.
?mkch "categoria" nome-canal Cria um canal de texto normal.
?mkchv "categoria" nome-canal Cria um canal de voz normal.
?mkpch "categoria" nome-canal Cria um canal de texto secreto.
?mkpchv "categoria" nome-canal Cria um canal de voz secreto.
?delch "categoria" Deleta uma categoria e todos os canais existentes na mesma.
?remove_roles ou ?rmr @user @role Remove uma regra @role do usuário @user.
?setprefix prefixo Muda o prefixo do bot (ex. ?setprefix p!)
?remind ou ?rd DD/MM/AAAA @user @role Adiciona um cargo temporário ao usuário até a data DD/MM/AAAA (Beta)
?addroles ou ?adr @oldrole @newrole Adiciona um cargo @newrole aos usuários que têm o cargo @oldrole.
?adduser_roles ou ?adur @user @role Adiciona uma regra @role ao usuário @user.
?wellcome #channel-name ou ?wcm #channel-name Define um canal como registro por reação. Você deve criar um canal somente para isso e um cargo com o nome Membro.
?reactionrole @cargo emoji ou ?rr @cargo emoji Cargos por reações. Você pode inserir até 10 cargos de uma vez.
?defchatguild #channel ou ?dfcg #channel Define um canal como chatguild. Com esse canal os donos e admins poderão se comunicar com outros donos e admins.
?udefchatguild #channel ou ?udcg #channel Define um canal como não sendo de chatguild.
?listguild ou ?lgd Lista os possíveis servidores para chatguild.
?chatguild ID-guild "Mensagem" ou ?cgd ID-guild "Mensagem" Envia uma mensagem para o servidor informado pela ID.
